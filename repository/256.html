MediaWiki 1.36.2 をダウンロード

これは zip ファイルです。 これは、追加のソフトウェアをインストールせずにほとんどのオペレーティング システム (OS) で抽出できます。通常、Windows ユーザーにお勧めします。

tarball (tar.gz ファイル) も提供しています。 PAX 形式でのバグがあるため、tar アーカイブ (.tar.gz ファイル) の圧縮解除には 7-Zip を使用しないことをお勧めします。 macOS 利用者は、The Unarchiver にバグがあり、tarball 内のファイル名を破損してしまうことにご注意ください。 代わりにアーカイブ ユーティリティも使用できます。

  • リリース ノート
  • インストール / アップグレード / 設定
  • 注: 最新の安定リリースは安定ブランチへのバグ修正をすべては含んでいません。最新のスナップショットもあります。

システム要件

MediaWiki には PHP 7.4.9 以降に加え、MySQL 5.5.8+、MariaDB、SQLite、PostgreSQL のデータベース ソフトウェアのいずれかが必要です。更なる情報については、システム要件、互換性の各ページをお読みください。

MediaWiki は PHP 7.3.0 - 7.3.18 および 7.4.0 - 7.4.8 までと互換性がありません (上流のバグのため)。 代わりに 7.3.19+ または 7.4.9+ を使用してください。 詳細情報は タスク T246594 を参照してください。
警告: MediaWiki はまだ PHP 8 と互換性がありません。 詳細情報は タスク T248925 を参照してください。
MediaWiki バージョン 1.32-1.34 および バージョン 1.30 以前の利用者へ: これらのバージョンはもうサポートされなくなりました。より新しい MediaWiki をダウンロードしてください:
  • MediaWiki 1.36.2 (.zip をダウンロード, .tar.gz をダウンロード) - 安定版
  • MediaWiki 1.35.4 (.zip をダウンロード, .tar.gz をダウンロード) - 現行の長期間サポート (LTS: long-term support)

Gitからのダウンロード

MediaWiki のアクティブな開発者は、MediaWiki ソフトウェアの最新版を Git からダウンロードすべきです。 Git リポジトリには、ソフトウェアの以前のバージョンも格納されていて、特定のリリースに切り替える (checkout する) ことができます。

署名のダウンロード

  • GNU Privacy Guard (GPG) でダウンロードを検証するための GPG セキュリティ署名:
    • MediaWiki 1.36.2 - 安定版
    • MediaWiki 1.35.4 - 現行の長期間サポート (LTS: long-term support)
    • MediaWiki 1.35.4 - 旧版の長期間サポート (LTS: long-term support)
  • GPG 鍵
  • MediaWiki 1.36.2 の変更点 (国際化を除く) (統合された差分)

注記

Microsoft SQL Server、Oracle Database と HHVM ならびに PHP 7.0、 7.1 はすでに MediaWiki 1.34 でサポート対象から除外されています。

手動インストールの代替手段

統合済みの MediaWiki ソフトウェア アプライアンスやホスティング サービスを使用したいユーザーもいるでしょう。一部の Linux ディストリビューションでは MediaWiki 向けのパッケージも提供していますが、更新頻度や拡張機能の収録範囲はそれぞれ異なります (例: Debian、Ubuntu、Fedora、Gentoo)。

旧版リリース

MediaWiki インストレーションを大きく変更している場合は、MediaWiki の最新の公式の変更/更新を反映させるのが難しい場合があります。そのような利用者を支援するために、旧版リリースを最大1年間、旧版 LTS リリースを最大3年間、コードの古いブランチを維持しています。

  • 旧版リリース MediaWiki をダウンロード (1.35.4)

開発者向けリリース

最新の開発版 (つまり、アルファ版) を使用したい場合は、mediawiki-master.tar.gz としてダウンロードするか、または Git からダウンロードしてください。

Git から以下のいずれかをダウンロードできます: 完全なリポジトリ (約 528 MiB) 

git clone https://gerrit.wikimedia.org/r/mediawiki/core.git

または、最新リビジョンのみ (約 85 MiB。これはよく shallow clone と呼ばれます: ダウンロード時間が短くてサイズが小さいです)。[1] 

git clone --depth 1 https://gerrit.wikimedia.org/r/mediawiki/core.git

ブラウザーで最新のソースコードを閲覧することもできます。

以前のリリース

MediaWiki ダウンロード アーカイブを見ると2005年3月 (最古は MediaWiki 1.3.11) までさかのぼって、MediaWiki のすべてのバージョンの tarball (拡張子 *.tar.gz) を探せます。

古いバージョンも含めてブラウズするには、Git リポジトリのタグを参照してください。

インストール支援

  • よくある質問: インストールと設定
  • mediawiki-l (メーリング リスト) は、支援を求めるための流量が多いメーリング リストです。

脚注

  1. ↑ 後に "git pull" を使ってあなたの shallow clone をアップグレードしたり、まだあなたのシステムにダウンロードされていない古いリビジョンを新しいリビジョンが参照したりする際に、これは参照問題を起こす可能性があります。そのような場合は、単により多い（depth 値を増やす）あるいはすべてのリビジョンを後でダウンロードするか、新しい shallow clone を作ります。
