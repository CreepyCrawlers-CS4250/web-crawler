日本とモンゴルの時差

日本とモンゴルの首都ウランバートルの時差は-1時間。日本が午前1時になるとウランバートルは同じ日付の0時となる。西部は時差-2時間、東部は日本との時差がない。

日本と首都ウランバートルの時差は-1時間

モンゴル時間の計算

日本時間から1時間戻す。マイナスとなる場合はさらに24から引く。例：日本時間0時-1=-1と計算して、さらに24-1=23時となり現地時間は前日の23時となる。

UTC+8

首都のウランバートルは協定世界時から8時間進んだ標準時。UTC+8と表記される。

UTC+7

モンゴル西部は協定世界時から7時間進んだ標準時。UTC+7と表記される。

UTC+9

モンゴル東部は協定世界時から9時間進んだ標準時。UTC+9と表記される。

時差の計算

日本とモンゴルの時差は-1時間。日本の時間からモンゴルの時間を調べるには、日本時間を1時間戻せばよい。

日本時間 - 1時間 = モンゴル時間

日本とモンゴルの時差は-1時間。日本の時間からモンゴルの時間を調べるには、日本時間を1時間戻せばよい。単純に引き算で求めることができるが、計算してマイナスとなる場合はさらに24から引くと分かりやすい。

例1：日本10時 - 1時間 = モンゴル9時
例2：日本0時 - 1時間 = モンゴル-1時 → 24-1=23 → モンゴルは前日の23時

逆にモンゴル時間から日本時間を調べるには、モンゴル時間を1時間進めればよい。覚えておくと、モンゴル旅行中に日本の時間が知りたいときに便利。計算して24を超える場合は、24を引くと分かりやすい。

例3：モンゴル10時 + 1時間 = 日本11時
例4：モンゴル24時 + 1時間 = 25時 → 25-24=1 → 日本1時*翌日の深夜
