おそらく、フリーの百科事典であるウィキペディアはご存知でしょう。おそらく似ているが異なる、ウィキ、ウィキメディア、MediaWiki の用語が混乱を引き起こしています。

これらの用語の詳細については、これが説明されている「Wikipedia、Wikimedia、MediaWiki、wikiの違い 」をお読みください。


全般的な概要

MediaWiki はフリーなサーバーサイドソフトウェアであり、GPL ライセンスで配布されます。1 日に何百万ヒットもあるような巨大なサイトで運用可能なように設計されています。

MediaWiki は MySQL などのデータベースにデータを記録し、PHP でそのデータの取り出し、整形を行うパワフルでスケーラビリティのある機能豊富な Wiki ソフトウェアです。

それぞれのページは MediaWiki のウィキテキストの書式で記述されており、 HTML や CSS の知識がなくても簡単にページを編集できます。

ページを編集して保存すると、MediaWiki はそれをデータベースに書き込みます。しかしその際に以前の版を削除しないため、悪戯やスパム行為があっても簡単に元に戻すことができるのです。MediaWiki は画像やマルチメディアファイルも扱うことができ、ファイルシステムに保存します。MediaWiki は多くのユーザーがアクセスする巨大なウィキ用にキャッシュに対応しており、プロキシサーバー ソフトウェアとも簡単に連携できます。 専用の拡張機能を使用すると、MediaWiki は構造化データを処理することもできます。

MediaWiki とは...

MediaWiki はウィキ ソフトウェアです。
ウィキが何か分からない場合は、先に進む前にこちらのウィキペディアの記事をお読みください。
MediaWiki はサーバー ソフトウェアです。
インターネットにソフトウェアを公開すると、バグもしくはセキュリティ問題が起きることがあります。security upgradesを続ける意思がない場合は MediaWiki をインストールしないで下さい。(購読申込みををしてセキュリティアップグレードの告知を受信するようお願いします)。
MediaWiki は、ウィキメディア財団のプロジェクト群のニーズに合わせて調整されています。
プログラムは、ウィキペディアおよび姉妹プロジェクト群のための巨大サーバーファーム上で動作することを第一に開発を続けています。多くの場合、機能やパフォーマンス、設定変更のしやすさや使いやすさなど、この観点から設計しています。あなたのニーズとは根本的に異なる場合、このソフトウェアは適していないかもしれません。
MediaWiki はフリーソフトウェアです。
いかなる種類の保証もせず保証書も提供されません。

ウィキテキストを試してみる

ページの編集は非常に簡単で、一時的に仮の文章をリリースできます。ウィキ内のページを一時的に破棄することさえ簡単です。プログラムの知識はまったく必要ありません。

実際にサンドボックスで試してみることをお薦めします。ただし、テストはサンドボックスでのみ行なってください。それ以外のページで行なった場合には、管理者にそのいたずら書きを厳しく対処される場合があります。

書式設定の簡単な例を cheat sheetでご覧ください。

どんなことが MediaWikiに向いていないか...

  • MediaWiki はオープンなコンテンツのために設計されたため、ウィキの特定の部分にアクセス制限をかけたい場合にはあまり向いていません。 社内のネットワークのように、関係者だけアクセスできる wiki をホストすることは可能です。
  • MediaWiki はウィキペディアのような高トラフィックなウェブサイトに適応するよう設計してあります。その目的に最適化したため、帯域よりもディスクスペースもしくはメモリが小さいせいで制限を受ける小規模のサイトに適さないことがあります。
  • フォーラムまたはブログ専用のソフトウェアの代わりに MediaWiki を使おうとお考えなら、通常は最適ではありません。

上記の目的であれば、もっとニーズに合う別のウィキ ソフトウェア (またはウィキ以外のソフトウェア) があるかもしれません。
