MediaWiki は共同作業と文書作成のプラットフォームであり、活気のあるコミュニティが提供しています

MediaWiki ソフトウェアは何万件ものウェブサイトや何千件もの企業や組織で使用されています。 これはウィキペディアやこのウェブサイトも支えています。 MediaWiki は、知識を収集・整理してそれを人々が利用できるようにするのを支援します。 強力、多言語対応、フリー、オープン、拡張可能、カスタマイズ可能で、信頼性があり、無料です。 詳細を学び、MediaWiki があなたに適しているかどうか確認してください。

MediaWiki のセットアップと実行

  • MediaWiki をダウンロード、インストール、設定する
  • 拡張機能をインストールし、機能を追加する
  • 問題が発生しましたか？ エラーと症状およびよくある質問を参照してください。
  • サーバーをお持ちではありませんか？ ホスティング サービスを参照してください。
  • 専門家の開発およびコンサルティングを受ける
  • MediaWiki 利害関係者利用者グループに参加する

MediaWiki の編集と使用

  • サイトの使用方法を学ぶ
  • ページの編集方法を学ぶ
  • 閲覧、編集、個人用カスタマイズの詳細を学ぶ

コードの開発と拡張

開発の初心者ですか？

  • ウィキメディア プロジェクトでコーディングを始めましょう
  • MediaWiki 外装を作る方法
  • MediaWiki またはその拡張機能のコードの実行方法

以前からの開発者ですか？

  • 操作 API やウェブ API を使用してコンテンツと触れ合う
  • 開発者向け文書を読む、開発者ハブを参照する

支援を受ける・貢献する

  • MediaWiki の問題の解決策が見つかりませんか？ サポートデスクでお尋ねください！
  • 翻訳者、設計者、説明文書執筆者、テスター、技術大使、…として参加する
  • ソフトウェアの誤動作の報告、機能の提案
